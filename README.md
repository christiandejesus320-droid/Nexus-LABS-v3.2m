NEXUS LABS: AI Skills Ecosystem

Este no es otro repositorio de prompts genéricos. Es el núcleo operativo de NEXUS LABS, diseñado para quienes ejecutan, venden y escalan negocios usando inteligencia artificial.

Aquí tienes 6 herramientas modulares (Skills) construidas para una sola cosa: resultados. Cada una transforma a Claude en un especialista senior capaz de auditar, diseñar, programar y vender con precisión quirúrgica.




🛠 Lo que hay dentro

1. Pepe (Master Skill)

El cerebro del sistema. Si no sabes por dónde empezar o tu proyecto está estancado, Pepe audita la situación y te da la ruta más corta hacia el dinero. Es el estratega que no te dice lo que quieres oír, sino lo que necesitas hacer.

2. NovaXCo Operator

Especializado en el mundo streetwear y e-commerce premium. Automatiza el análisis de competencia, define estrategias de ventas y gestiona la operación diaria para que tú te enfoques en escalar.

3. Christian Marketing

El motor visual y creativo. Genera carruseles de alto impacto y descripciones técnicas para modelos de imagen (FLUX) que mantienen una estética de agencia de lujo. Marketing que convierte, no solo que se ve bien.

4. Web Master

Desarrollo web sin fricciones. Olvida las webs lentas o mal optimizadas. Esta skill asegura que cada línea de código cumpla con estándares de performance (Lighthouse 90+) y SEO técnico real.

5. UI/UX Pro Max

Inteligencia de diseño aplicada. No hacemos cosas "bonitas", hacemos interfaces que funcionan. Sistemas de diseño, jerarquía visual y accesibilidad para aplicaciones que los usuarios realmente aman usar.

6. LLM Council

Orquestación de modelos. Cuando una respuesta no puede fallar, esta herramienta hace que varios modelos (GPT-4, Claude, Gemini) debatan hasta llegar a la mejor solución técnica posible.




🚀 Cómo poner esto a trabajar

Para usar estas skills en Claude Code o cualquier entorno compatible:

1.

2.

•

•
"Ejecuta Christian Marketing: Necesito 3 hooks para este producto."

•
"Inicia Web Master: Revisa este código y hazlo volar."
